```
  exposure at that point is unavoidable.

  The cave system predates WDLK operations. It is not
  a WDLK asset. 0034-C's manifestation occupies it

  dispute. Pollution Man's placement there is

  Clearing the smog permanently would require beating

  PRIORITY:

  Restore Captain Planet to the KIOSK-03 terminal to
  re-contain the smog to the cave interior. This is
  being handled as part of Osei's full site rebuild.
```

```
  inspection from exterior shows no void beneath the
  building. The basement is not hidden. It is absent.

  847 Lakeshore Drive:
  Basement access remains via interior stairwell.
  Layout is reduced. The finished basement apartment
  — bedroom, bathroom, and connecting hall — is
  absent while game is running. Remaining space is a
  single open room consistent with an unfinished
  basement. Walls where the bedroom and bathroom
  entries should be are poured concrete, continuous,
  no seams.

  CURRENT STATUS (site offline since 02/15):

  With the game no longer running, both spaces have
  returned to their original, unmodified state.

  14200: Basement door, stairwell, and full structure
  are present and accessible.

  847: Full basement apartment is present — bedroom,
  bathroom, connecting hall, all finished.

  ASSESSMENT:

  0034-D does not produce a dangerous manifestation.
  It produces an absence. While running, it removes
  spaces. While offline, the spaces return.

  This has been operationally useful. The game has
  been used since its deployment on the original
  Kids Zone site to deny access to both basements.
  The station basement contains materials relevant to
  ongoing operations. The 847 basement apartment is
  part of the property cycle and its visibility
  creates complications during tenant turnover.

  RECOMMENDATION:

  Restore Escape Basement World to KIOSK-03 as part
  of the site rebuild. Resume suppression of both
  basement spaces. The current exposed state of the
  847 basement is a liability if the property goes
  to market before the game is restored.

  NOTE:

  Deliberately using 0034-D as a concealment tool is
  a policy decision, not a containment decision. The
  asset is not dangerous when unbound — the spaces
  simply come back. Restoring the game specifically
  to hide the basements is an active operational
  choice. This should be acknowledged and authorized
  by Calloway.

================================================================
```

```
INTERNAL MEMORANDUM — WDLK BROADCASTING INC.

DATE:    05/01/1996
FROM:    K. Osei, Production
TO:      D. Rourke
RE:      0034-C — gameplay status

Dale,

I've been trying to clear Captain Planet since the
restore. Eight stages. I've cleared seven. Stage eight
is the Pollution Man boss fight, three phases. I can
get through the first two. Phase three is the problem.

The health pickups in stages four and six
aren't spawning. They're in Vasquez's source code. The
game just isn't placing them.

Best run: Stage eight with 1HP. Not enough. His spray
pattern covers too much screen. You need at least 3 to
absorb the unavoidable overlaps.

The save file is now locked at the final level with 1HP. I
can't start a new game. The reset function exists in
the code but the game won't execute it. It loads the
save every time.

We're locked out.

If clearing the game is the only way to permanently
clear the smog and access the Origin site, then the
Origin site is inaccessible. The game won't let us
win.

                                        — K.
```
